# Installation

Add Widget Tooltip to your Flutter project by following these simple steps:

## 1. Add Dependency

Add the following line to your project's `pubspec.yaml` file under `dependencies`:

```yaml
dependencies:
  widget_tooltip: ^1.0.0  # Replace with the latest version
```

Or run this command in your terminal:

```bash
flutter pub add widget_tooltip
```

## 2. Import Package

Import the package in your Dart code:

```dart
import 'package:widget_tooltip/widget_tooltip.dart';
```

## 3. Run Flutter Pub Get

If you manually edited `pubspec.yaml`, run:

```bash
flutter pub get
```

## Platform Support
