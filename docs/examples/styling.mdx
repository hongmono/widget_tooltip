# Styling

Widget Tooltip offers extensive styling options to customize the appearance of your tooltips. This guide demonstrates various styling techniques and examples.

## Basic Styling

### Message Box Styling

Customize the tooltip container's appearance:

```dart
WidgetTooltip(
  message: Text('Styled tooltip'),
  child: Icon(Icons.palette),
  messageDecoration: BoxDecoration(
    color: Colors.blue,
    borderRadius: BorderRadius.circular(8),
    boxShadow: [
      BoxShadow(
        color: Colors.black.withOpacity(0.2),
        blurRadius: 4,
        offset: Offset(0, 2),
      ),
    ],
  ),
  messagePadding: EdgeInsets.symmetric(
    horizontal: 16,
    vertical: 8,
  ),
)
```

### Triangle Styling

Customize the pointing triangle:

```dart
WidgetTooltip(
  message: Text('Custom triangle'),
  child: Icon(Icons.arrow_drop_down),
  triangleColor: Colors.purple,
  triangleSize: Size(15, 15),
  triangleRadius: 4,
)
```

### Text Styling

Style the tooltip text:

```dart
WidgetTooltip(
  message: Text(
    'Styled text tooltip',
    style: TextStyle(
      color: Colors.white,
      fontSize: 16,
      fontWeight: FontWeight.bold,
      letterSpacing: 0.5,
    ),
  ),
  child: Icon(Icons.text_fields),
)
```

## Advanced Styling

### Custom Widget Content

Use any widget as tooltip content:

```dart
WidgetTooltip(
  message: Container(
    constraints: BoxConstraints(maxWidth: 200),
    child: Column(
      mainAxisSize: MainAxisSize.min,
      children: [
        Image.asset(
          'assets/example.png',
          height: 100,
          width: 100,
          fit: BoxFit.cover,
        ),
        SizedBox(height: 8),
        Text(
          'Custom tooltip with image',
          style: TextStyle(
            color: Colors.white,
            fontSize: 14,
          ),
        ),
        Divider(color: Colors.white.withOpacity(0.5)),
        Text(
          'Additional details here',
          style: TextStyle(
            color: Colors.white.withOpacity(0.8),
            fontSize: 12,
          ),
        ),
      ],
    ),
  ),
  child: Icon(Icons.image),
  messageDecoration: BoxDecoration(
    color: Colors.black.withOpacity(0.9),
    borderRadius: BorderRadius.circular(12),
  ),
)
```

### Gradient Background

Apply gradient backgrounds:

```dart
WidgetTooltip(
  message: Text(
    'Gradient tooltip',
    style: TextStyle(
      color: Colors.white,
      fontSize: 16,
    ),
  ),
  child: Icon(Icons.gradient),
  messageDecoration: BoxDecoration(
    gradient: LinearGradient(
      colors: [Colors.purple, Colors.blue],
      begin: Alignment.topLeft,
      end: Alignment.bottomRight,
    ),
    borderRadius: BorderRadius.circular(8),
  ),
  triangleColor: Colors.purple,
)
```

### Interactive Content

Create interactive tooltips:

```dart
WidgetTooltip(
  message: Container(
    padding: EdgeInsets.all(8),
    child: Column(
      mainAxisSize: MainAxisSize.min,
      children: [
        Text(
          'Interactive Tooltip',
          style: TextStyle(
            color: Colors.white,
            fontSize: 16,
            fontWeight: FontWeight.bold,
          ),
        ),
        SizedBox(height: 8),
        Row(
          mainAxisSize: MainAxisSize.min,
          children: [
            ElevatedButton(
              onPressed: () {
                // Handle action
              },
              child: Text('Action 1'),
              style: ElevatedButton.styleFrom(
                primary: Colors.blue,
                onPrimary: Colors.white,
              ),
            ),
            SizedBox(width: 8),
            ElevatedButton(
              onPressed: () {
                // Handle action
              },
              child: Text('Action 2'),
              style: ElevatedButton.styleFrom(
                primary: Colors.green,
                onPrimary: Colors.white,
              ),
            ),
          ],
        ),
      ],
    ),
  ),
  child: Icon(Icons.touch_app),
  messageDecoration: BoxDecoration(
    color: Colors.grey[800],
    borderRadius: BorderRadius.circular(12),
    boxShadow: [
      BoxShadow(
        color: Colors.black.withOpacity(0.3),
        blurRadius: 8,
        offset: Offset(0, 4),
      ),
    ],
  ),
)
```

## Theme Integration

Integrate with your app's theme:

```dart
class ThemedTooltip extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    final theme = Theme.of(context);
    
    return WidgetTooltip(
      message: Text(
        'Theme-aware tooltip',
        style: theme.textTheme.bodyText2?.copyWith(
          color: theme.colorScheme.onPrimary,
        ),
      ),
      child: Icon(Icons.palette),
      messageDecoration: BoxDecoration(
        color: theme.colorScheme.primary,
        borderRadius: BorderRadius.circular(8),
      ),
      triangleColor: theme.colorScheme.primary,
    );
  }
}
```

## Styling Best Practices

1. **Consistency**
   - Maintain consistent styling across your app
   - Use theme colors and text styles
   - Keep padding and border radius consistent

2. **Readability**
   - Ensure sufficient contrast
   - Use appropriate text sizes
   - Consider different screen sizes

3. **Performance**
   - Avoid complex widget trees in tooltips
   - Use const constructors where possible
   - Minimize expensive decorations

4. **Accessibility**
   - Provide sufficient touch targets
   - Ensure text is readable
   - Consider color-blind users
