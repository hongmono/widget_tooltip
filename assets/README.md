# Assets

이 디렉토리는 README 파일에 포함될 이미지와 데모 자료를 포함합니다.

## 필요한 이미지

### demo.gif
Widget Tooltip의 동작을 보여주는 애니메이션 GIF 파일입니다.

**권장 사항:**
- 크기: 800x600px 또는 유사한 비율
- 포맷: GIF (애니메이션)
- 내용: 다양한 트리거 모드와 방향을 보여주는 데모

**생성 방법:**
1. 예제 앱 실행: `cd example && flutter run`
2. 화면 녹화 도구 사용 (LICEcap, ScreenToGif, Kap 등)
3. 다음 기능들을 보여주는 시연:
   - 탭으로 툴팁 표시
   - 롱 프레스로 툴팁 표시
   - 다양한 방향 (위, 아래, 왼쪽, 오른쪽)
   - 스타일링 예제
   - 엣지 감지 동작
